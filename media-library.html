The JavaScript Media Library

This was a one month project we had to do during our free time.

One of the aspect of the Application

How did I do it ?

During our JavaScript formation, we had a projet, and it was to do a only JavaScript, as the subject's course is about, a media library.

First version

I started by doing a rapid mockup of what I wanted to do, during the short month I had.

After that, I did the first version of it, which contained only the display of the different media, in the way I wished it.

When the display was done, I added the button to remove them, and add one media, with a modal.

The majority of the display is done with JavaScript, every media is created by it. When the media is created, it is added to an array, which simplify the removing after this.

Using LocalStorage

To store Media, without using a database, I'm using LocalStorage, which stores it in the browser, like this, if you do not restart it, you cannot lose your Media.

Searching by Type

When you want to search by Type, you can do that by clicking on the navbar, on the type you want to search by.
